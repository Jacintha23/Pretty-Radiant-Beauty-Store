# PRETTY RADIANT BEAUTY STORE
This is a simple point of sales application that allows beauty consumers to build a beauty order of foundation. They are given 6 customization options
(Quality, Size, Shade, Form, Finish, Skin Type), including a specialized fragrance add-on. They also recieve an order summary for verification and, 
a receipt of their beauty order.
